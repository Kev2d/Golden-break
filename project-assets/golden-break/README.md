# Project Template README

## Overview
This is a base template for web development projects. It includes a set of folders and files that are commonly used as a starting point.

## Folders
- `acf-blocks/`: Blocks for Advanced Custom Fields.
- `assets/`: Images, fonts, JS, and SCSS files.
- `functions/`: PHP functions and custom WordPress hooks.
- `include/`: External libraries and scripts.
- `template-parts/`: WordPress template parts.
- `node_modules/`: Node.js modules (not to be uploaded to version control).

## Setup
1. Clone the repo.
2. Run `npm install` to set up node modules.
3. Modify PHP functions as needed.
4. Add custom post types.
5. Place front-end assets in the `assets/` folder.

## Usage
Customize the files and folders as needed for the project.

## Contribution
Feel free to fork and use this template for your projects.
